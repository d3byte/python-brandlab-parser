#успех

По-моему мнению, для каждого человека успех - понятие относительное. Для кого-то успехом будет миллион  заработанный денег от собственного бизнеса, а для кого-то просто приятный отзыв от близкого человека. Для кого-то будет успехом - получение кубка мира по гимнастике, а для кого - то шпагат сделанный после мучительной растяжки. (очень актуальная тема после сегодняшнего карате, кстати)))
